Preface

This is a tutorial and set of working examples for creating web-based textbooks using a collection of open-source tools.

This web-book is itself a working example. All of the source code needed to compile this book yourself is included in the github repository for this book. So, you could download the repository, and by following the instructions laid out across the chapters, replace this text with your own, and then compile your book as a web-page, .pdf or epub.
